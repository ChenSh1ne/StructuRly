
1. Install all StructuRly dependencies:

install.packages(pkgs = c("ade4", "adegenet", "assertthat", "colourpicker", "dendextend", "ggplot2", "grDevices", "janitor", "mcclust", "pegas", "plotly", "poppr", "processx", "randomcoloR", "RColorBrewer", "reshape2", "scales", "shiny", "shinyjs", "tidyr"))



2. At the question "Do you want to install from sources the package which needs compilation?" type "no" and wait until all dependencies are installed.



3. Install StructuRly with the following line command:

install_github(repo = "nicocriscuolo/StructuRly", dependencies = TRUE)



4. Decide whether to update dependencies or not and wait until StructuRly is installed.
